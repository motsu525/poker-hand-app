# 1. サービス概要

- ライブポーカーのハンド履歴を簡単に入力できるサービス
- 入力したデータを分析ツール(※GTOWizard)に即連携できる
- 振り返りのハードルを下げ、プレイヤーの上達を支援する
※ポーカーにおけるGTO（理論最適戦略）を学習・分析できるツール
　以下、GTOと記載。

# 2. このアイデアはどこから生まれたか

## 2-1. きっかけとなった体験・感情

- いつ：ライブポーカーをプレイした直後
- 状況：プレイ後にハンドを振り返ろうとしたが、記録が曖昧
- 気持ち：
  - 「あのハンドちゃんと検証したいのに思い出せない」
  - 「メモするの面倒で結局やらない」
  - 「GTOで分析したいのに入力がダルい」

## 2-2. なぜそれが気になったのか

- ライブポーカーは人と人で行う都合上、気になったハンドがあっても振り返る時間が少ない
- GTOは解析には優れているが、色々な状況の解析ができる一方、その状況を入力するのが手間。
※なぜ入力が手間なのか
→ライブポーカーでは複数人でプレイすることが一般的だが、GTOツールで分析したいのは主に「自分が参加したハンド」。しかし、GTOツールでは各プレイヤーのアクションをすべて手動で入力する必要があり、実際には多くのプレイヤーが序盤でフォールドしているにもかかわらず、その入力も省略できない点が手間となっている。

- 「改善できるはずの機会を逃している感覚」がストレスだった

# 3. 課題の整理

## 3-1. 表に見えている困りごと

- ハンド履歴を正確に記録するのが面倒
- GTOに入力するのが手間

## 3-2. 本当に解決したい課題

- ハンド履歴を気軽に入力、かつ履歴として残したい
※ どのように入力の手間を減らすのか
→「自分が参加したハンドの分析」に焦点を当て、自分のハンドと実際にアクションに関与したプレイヤーのみを入力することで、その他のプレイヤーのアクションを自動的に補完し、GTOツールに取り込める形式のテキストを生成することで入力工数を削減する

- メモを取るやり方だと時間がかかるし、GTOにも別途入力する必要があり二度手間

→ 本当に解決したい課題：
「振り返りたいのに、面倒さによって行動できない状態」


# 4. 想定ユーザー

## 4-1. 想定しているユーザー

- ライブポーカーを定期的にプレイ
- セッション後に振り返りをしたい
- 理手入力より欲があるが、記録が続かない

環境：
- カジノ・アミューズメント施設
- 相手あってプレイできる環境なので自身の振り返りに割ける時間は限られている
- わずかな時間で入力が可能かつ、あとで簡単に連携できることで時間の制約を解決する

## 4-2. 自分とユーザーの距離

→ 自分自身

理由：
- 自分がまさに困っている
- 解像度の高い課題理解がある

## 4-3. 実際に使われる可能性

- 最初のユーザー：自分 + ポーカー仲間
- 利用シーン：プレイ直後 and 帰宅後
- 代替手段：
  - メモ帳
  - 頭で記憶
  - 入力を諦める
- 置き換え理由：
  - 圧倒的に楽なら使う
  - ワンクリックで分析に繋がるなら価値あり

## 4-4. 導入・継続の流れ

- きっかけ：
  - 「振り返りたいけど面倒」
- 初回：
  - ログイン → ハンド入力 → コピー
- 継続理由：
  - 手入力より簡単に入力が可能
  - 記録が蓄積される
  - 分析への連携も簡単
